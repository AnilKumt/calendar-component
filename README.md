# Calendar Component

Live link: https://calender-component-tuf.netlify.app/

## Features

- Wall clock with spiral binding, 3D turn effect, and smooth transitions.
- Holiday dates highlighted with pins.
- Theme toggle.
- Light and dark mode support.
- Date range selection with highlighted ranges.
- General monthly notes and notes for selected date ranges.
- Responsive layout for all devices.
- Modular code structure.

## Notes

The app is built with a focus on clean composition, reusable sections, and a responsive calendar experience.
